1.3 Interfaces de entrada e saída

As interfaces de entrada e saída (E/S) são componentes cruciais em sistemas computacionais, facilitando a comunicação entre a CPU e os dispositivos periféricos. Permitem que o computador receba dados de dispositivos de entrada (como teclados e mouses) e envie dados para dispositivos de saída (como monitores e impressoras). Essencialmente, as interfaces de E/S atuam como um "tradutor" entre a linguagem interna do computador e a dos dispositivos externos.

1.3.1 Interfaces sem fio

Uma interface sem fio, ou interface de rede sem fio (Wireless Network Interface - WNIC), é um dispositivo que permite a comunicação sem fio entre dispositivos eletrônicos, utilizando tecnologias como Wi-Fi, Bluetooth ou Zigbee, por exemplo. Ela permite a conexão de dispositivos a uma rede sem a necessidade de cabos físicos.

1.3.3 Rede (Ethernet)

Uma rede Ethernet é uma tecnologia de rede de computadores com fio amplamente utilizada para conectar dispositivos em redes locais (LAN), redes metropolitanas (MAN) e redes de longa distância (WAN). Funciona transmitindo dados através de cabos de rede, garantindo uma comunicação segura e rápida entre os dispositivos conectados.

1.3.4 VGA

VGA é a sigla para "Video Graphics Array", um padrão de exibição de vídeo e tipo de conexão amplamente utilizado em computadores. Introduzido pela IBM em 1987, o VGA tornou-se o padrão gráfico padrão para PCs e lançou as bases para os monitores de computador modernos.

1.3.5 HDMI

HDMI (High-Definition Multimedia Interface) é uma tecnologia de conexão por cabo que transmite vídeo e áudio de computadores, celulares e tablets a TVs, monitores ou projetores. A tecnologia pode transmitir conteúdo em resoluções como 4K e 8K.

1.3.6 USB

é um padrão de conexão por cabo que permite transmissão de dados entre dispositivos variados, como celulares, PCs, TVs e acessórios. As especificações da tecnologia também permitem que uma conexão USB forneça alimentação elétrica ou faça recarga de bateria.
